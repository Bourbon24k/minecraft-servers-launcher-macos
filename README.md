# Minecraft macOS Console Launcher

Консольный лаунчер для создания, настройки, запуска и управления одним или несколькими Minecraft-серверами на macOS.

## Быстрый старт

1. Запустите лаунчер:

   ```zsh
   ./minecraft_launcher.py
   ```

   Или откройте двойным кликом файл `Start Minecraft Launcher.command`.

2. В меню выберите `Создать сервер`.

3. В выборе Java оставьте пункт `Скачать автоматически Eclipse Temurin`.

Лаунчер сам скачает подходящую Java в `.mc-launcher/java/`, не устанавливая ее глобально в macOS.

## Ручная загрузка Java

Можно заранее скачать JDK:

   ```zsh
   ./minecraft_launcher.py java install 21
   ./minecraft_launcher.py java install 25
   ```

Для Minecraft 1.20.5+ обычно нужна Java 21, для ветки 26.x - Java 25. При создании сервера лаунчер выбирает версию автоматически.

## Что умеет

- создавать серверы Paper, Purpur, Fabric, Forge, vanilla или подключать свой `server.jar`;
- скачивать ядро самостоятельно по выбранной версии или `latest`;
- скачивать Eclipse Temurin JDK для macOS автоматически;
- устанавливать Forge через официальный installer в headless-режиме;
- хранить несколько серверов в папке `servers/`;
- настраивать RAM, порт, Java, JVM args, EULA и основные `server.properties`;
- запускать несколько серверов одновременно из одного лаунчера;
- отправлять команды в консоль сервера: `say`, `op`, `whitelist`, `stop` и другие;
- показывать логи в отдельном экране, не смешивая их с главным меню;
- открывать live-tail логов с ручным выходом через `q`;
- показывать LAN-IP, внешний IP и порт для подключения из домашней сети;
- открывать папку выбранного сервера в Finder;
- писать live-логи в `.mc-launcher/logs/`.

## Поддерживаемые ядра

```zsh
./minecraft_launcher.py cores
```

- `Paper` - плагины Bukkit/Spigot/Paper, хорошая производительность.
- `Purpur` - форк Paper с большим количеством настроек.
- `Fabric` - сервер под Fabric-моды.
- `Forge` - сервер под Forge-моды, ставится через Forge installer.
- `Vanilla` - официальный сервер Mojang.
- `Custom` - ваш локальный `server.jar`.

## Домашняя сеть и белый IP

В меню есть пункт `Сеть и адрес подключения`. Он покажет:

- LAN-IP этого Mac;
- внешний IP;
- порт выбранного сервера;
- адрес для игроков в домашней сети;
- адрес для игроков из интернета.

Для доступа из интернета при белом IP пробросьте на роутере TCP-порт сервера, например `25565`, на LAN-IP этого Mac. Если включен macOS Firewall, разрешите входящие подключения для Java.

## Важная деталь

Управление запущенными серверами работает, пока открыт сам лаунчер. Он держит Java-процессы как дочерние процессы и отправляет команды в их stdin. При выходе лаунчер предложит корректно остановить все запущенные серверы.

Логи серверов не печатаются поверх главного меню. Откройте пункт `Логи и консоль сервера`, чтобы посмотреть последние строки, включить live-логи или отправить команду.

## Файлы проекта

- `minecraft_launcher.py` - основной лаунчер.
- `Start Minecraft Launcher.command` - запускатель для macOS Terminal.
- `.mc-launcher/config.json` - конфиг серверов, создается автоматически.
- `servers/<name>/` - папки Minecraft-серверов.

## Проверка Java

```zsh
./minecraft_launcher.py java
```

Если системная Java не находится, это не проблема: скачайте ее через меню `Показать Java` или командой `./minecraft_launcher.py java install 21`. В настройках сервера можно назначить скачанную Java, системную Java или свой путь к `bin/java`.
